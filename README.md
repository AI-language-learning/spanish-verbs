# スペイン語動詞 Web資料 v31

v30 で発生していた表示崩れへの抜本対応版。

## v31 修正

- 既存 CSS の重ね書きと `!important` 連発を廃止し、`style.css` を全面的に整理。
- 動詞変化表の内部セル罫線を削除。
- 動詞変化表のフォントサイズを小さめに固定し、長い形でも途中改行しにくい設定に変更。
- `estudiábamos` などの単語が語中で改行されないように `white-space: nowrap` / `word-break: keep-all` / `hyphens: none` を指定。
- 強調表示の太字を解除。語尾の赤、補助動詞の青は色だけ残した。
- 欧文フォントは Mac で読みやすい `Avenir Next` を優先し、`Helvetica Neue`、`Segoe UI`、`Arial` にフォールバック。
- 主語６枠の配色は PDF の配色に合わせ、yo/nosotros 系＝薄紫、tú/vosotros 系＝ミント、él/ellos 系＝ピーチに整理。
- 全体の見た目を軽くするため、外枠・影・余白・背景色を再調整。

## v31b 微調整

- 全体フォントをさらに縮小。
- 主語表セルの疑似罫線も削除。
- 動詞表の2列幅を少し広げ、haber などの長い形でも重なりにくくした。
