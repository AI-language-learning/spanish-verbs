## モード表示順

各動詞の変化表は2列3段で表示する。

左列:
1. 「今・ふだん・ちょっと先」モード
2. 「今しているところ」モード
3. 「こうしてほしい・こうだといい」モード

右列:
1. 「もうした・したことがある」モード
2. 「できごとの過去」モード
3. 「情景の過去」モード

スマホ幅では、左列3つのあとに右列3つが続く1列表示になる。
